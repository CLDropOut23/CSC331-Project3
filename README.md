# CSC331-Project3
Project #3 for CSC331. This project uses a Java based Direct-Map Caches Simulator that checks the miss rates of different size Caches and Blocks.

For this project, we are using a Cache Simulating Code that was created in Java.  This code evaluates the performance of Caches and checks the miss rates of different Cache and Block sizes.
